Why do you decide to log these actions?

              Logging user activities on site gives a deeper understanding of customers. It helps us to explore the hidden uncover pattern, trends, correlation and other information about the data. Based on those findings we can craft the rich experiences with progressive enhancement for each user.  Considering the importance of this, I decided to log following activities of user.

  1. Click on Question: When user clicks on any question, I stored the question text, votes/views/answers on question, tags associated with question.
  2. Upvote question: On clicking the upvote button, I logged the question text, votes/views/favorites on question, tags associated with question. This log event helps us understanding the likings of the user, aid us recommending questions like given question based on tags associated with it.
  3. Downvote question: On clicking the downvote button, I logged the question text, votes/views/favorites on question, tags associated with question. Unlike upvote question button, this event helps us to understand dislikes of user.
  4. Favorite question: On clicking the favorite button, I logged the question text, votes/views/favorites on question, tags associated with question. Just like the upvote button, this helps us understand the likes of user. In addition, this event should be given more weightage than upvote event because by bookmarking a question, it tells us that user may want to see this question in future.
  5. Un-favorite question: On clicking the Un-favorite button, I logged the question text, votes/views/favorites on question, tags associated with question. This event tells us that user is now no more interested in such questions and we can minimize the question related to this in user’s feed.

In addition to the above 5 user behavior, I also logged the tag click, share question, and page change events of user. These all user actions helps us to identify the likes and dislikes of user, so that we can give personalized user feedback on the website.

 

 

Analysis & Findings:

By visualizing the collected the data, I found the following patterns:

  1. JAVA is the most popular technology among all user, though individual user may have different likings.
  2. Most common user reaction to any question is upvoting it(around 50%). Un-favorite the question is the least.
  3. Most users only view the question and only around 5% of them react with any activity (upvote, downvote, favorite, unfavorite) to that question.

To find these patterns I created the Horizontal bar chart which display the question view count of top 10 popular tags for individual user and all user. On hovering of any bar, a donut chart appears which represents the user activity on that tag. In the center of a chart, statistics of that chart is displayed i.e. Tag name, views on tag, and user engagement on that tag. User engagement tells us that how active the user is to provide any feedback to any question by upvote/downvote/(un)favorite it. It is the ratio of user’s feedback on that question and user’s view on that question.

Length of each bar chart provides the pattern of how many times a tag is being view. Proportion of each activity on donut chart explains the how much it is compared to other activities.

All charts are shown for both individual user and all users. This social visualization try to progress the user induvial user behavior among the rest of the community and how it is impacting the community.
